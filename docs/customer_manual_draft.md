## 1. はじめに

### システム概要
配置管理システムは、日々の案件に対してスタッフを配置し、LINE連絡用のテンプレートを作成するためのWebアプリケーションです。

### 画面構成
画面上部のナビゲーションバーから各ページに移動できます。

| メニュー | 説明 |
|---------|------|
| ダッシュボード | 当日の案件を時間帯別に一覧表示（メイン画面） |
| 案件一覧 | 期間を指定して案件を検索・管理 |
| 配置管理 | 配置状況の一覧表示 |
| 顧客 | 顧客情報の登録・管理 |
| スタッフ | スタッフ情報の登録・管理 |
| 設定 | 外注先・交通費・自社情報 |

### 推奨ブラウザ
- Google Chrome（最新版）
- Microsoft Edge（最新版）

## 2. ダッシュボード

ダッシュボードは本システムのメイン画面です。選択した日付の案件が時間帯別に表示されます。

【スクリーンショット: ダッシュボード全体】

### 2.1 画面構成

#### 日付ナビゲーション
- **< 前日**: 前日の案件を表示
- **翌日 >**: 翌日の案件を表示
- **今日**: 本日に戻る
- **統計**: 総件数・配置済・未配置の件数を表示

#### 時間帯カラム
案件は以下の時間帯に分類されます：
- **上棟**: 上棟作業
- **終日**: 1日を通しての作業
- **AM**: 午前中の作業
- **PM**: 午後の作業
- **夜勤**: 夜間作業
- **未定**: 時間帯未確定

#### 統計表示
各カラムのヘッダーに以下が表示されます：
- **件数**: その時間帯の案件数
- **過不足**: 配置人数の過不足（例: 「+4名余裕」「3名不足」）

### 2.2 案件カードの見方

【スクリーンショット: 案件カード拡大】

各案件は以下の情報を表示します：
- **種別タグ**: 鳶（赤）/ 揚げ（青）/ 鳶揚げ（緑）
- **顧客名**: 案件の顧客
- **配置人数**: 配置済/必要人数（例: 2/3）
- **現場名**: 作業現場の名称
- **住所**: 現場住所
- **開始時間**: 作業開始時間

※ 人数不足の案件は赤枠で強調表示されます。

### 2.3 案件カードの操作

- **カードをクリック**: 配置モーダルを開く（スタッフ配置画面）
- **鉛筆アイコン**: 案件編集モーダルを開く

## 3. 案件の登録・編集

### 3.1 新規案件の登録

1. ダッシュボード右下の「**+**」ボタン（丸いボタン）をクリック
2. 案件情報を入力

【スクリーンショット: 案件登録モーダル】

#### 入力項目

| 項目 | 必須 | 説明 |
|------|:----:|------|
| 顧客 | ○ | 顧客名を入力（候補から選択可） |
| 現場名 | ○ | 作業現場の名称 |
| 現場住所 | - | 作業現場の住所 |
| 作業日 | ○ | 作業を行う日付 |
| 時間帯 | ○ | 上棟/終日/AM/PM/夜勤/未定 |
| 開始時間 | - | 作業開始時間（例: 08:00） |
| 必要人数 | ○ | 配置が必要なスタッフ数 |
| 作業種別 | ○ | 鳶/揚げ/鳶揚げ |
| ステータス | - | 未配置/保留 |
| 監督者名 | - | 現場監督の名前 |
| 備考 | - | 特記事項 |

3. 「**保存**」ボタンをクリック

#### 連続登録
複数の案件を続けて登録する場合：
1. 「**保存して次へ**」ボタンをクリック
2. フォームがリセットされ、続けて次の案件を入力できます

### 3.2 案件の編集

1. ダッシュボードで編集したい案件の**鉛筆アイコン**をクリック
2. 内容を修正
3. 「**保存**」ボタンをクリック

### 3.3 案件のステータス

| ステータス | 説明 |
|-----------|------|
| 未配置 | スタッフ配置待ち（初期状態） |
| 配置済 | スタッフ配置完了 |
| 保留 | 一時停止中（顧客都合等） |
| キャンセル | 案件中止 |

## 4. スタッフの配置

### 4.1 配置モーダルを開く

ダッシュボードで案件カードをクリックすると、配置モーダルが開きます。

【スクリーンショット: 配置モーダル】

### 4.2 画面構成

#### スタッフ一覧（左パネル）
- 配置済みスタッフが上部に表示（緑色背景）
- 未配置スタッフが下部に表示
- チェックボックスでスタッフを選択

#### 絞り込み機能
- **名前検索**: 名前でスタッフを検索
- **スキルフィルター**: 鳶/揚げ/鳶揚げ
- **種別フィルター**: 自社/学生/個人事業主/外注

#### 配置設定（右パネル）
| 項目 | 説明 |
|------|------|
| 給与区分 | 終日/ハーフ |
| 請求区分 | 終日/ハーフ |
| 交通費エリア | 登録済みエリアから選択 |
| 交通費 | 金額（円） |
| 役割 | 鳶/荷揚げ/鳶揚げ |
| リーダー | リーダーに指定するかどうか |

### 4.3 スタッフを配置する

1. スタッフ一覧から配置したいスタッフにチェック
2. 右パネルで給与区分・請求区分・役割などを設定
3. 「**変更を登録**」ボタンをクリック

### 4.4 配置を解除する

1. 配置済みスタッフの「**×**」ボタンをクリック
2. 「**変更を登録**」ボタンをクリック

---

## 5. LINEテンプレート

配置が完了したら、スタッフへの連絡用にLINEテンプレートを生成できます。

### 5.1 テンプレートの生成

1. 配置モーダルで「**LINEテンプレ**」ボタンをクリック
2. テンプレート編集画面が開きます

【スクリーンショット: LINEテンプレート編集】

### 5.2 テンプレートの編集

生成されたテンプレートは自由に編集できます。
- 追加の連絡事項
- 現場への行き方
- 持ち物の指示 など

### 5.3 テンプレートのコピー

1. 「**コピー**」ボタンをクリック
2. クリップボードにコピーされます
3. LINEアプリに貼り付けて送信

## 6. 案件一覧

登録された案件を一覧で確認・管理できます。

【スクリーンショット: 案件一覧画面】

### 6.1 案件の検索・絞り込み

以下の条件で案件を絞り込むことができます：

| 項目 | 説明 |
|------|------|
| 作業日 | 指定日以降の案件を表示 |
| 時間 | 時間帯（上棟/終日/AM/PM/夜勤/未定）で絞り込み |
| ステータス | 案件の状態（未配置/配置済/保留/完了/キャンセル）で絞り込み |
| 検索 | 顧客名・現場名で検索 |

### 6.2 表示項目

| 項目 | 説明 |
|------|------|
| 作業日 | 案件の作業日 |
| 時間帯 | 上棟/終日/AM/PM/夜勤/未定 |
| 顧客名 | 顧客の名前 |
| 現場名 | 作業現場 |
| 種別 | 鳶/揚げ/鳶揚げ |
| 人数 | 配置済/必要人数 |
| ステータス | 案件の状態 |

### 6.3 ソート

テーブルヘッダーの「作業日」をクリックすると、ソート順を切り替えられます。

## 7. 配置管理ページ

ナビゲーションの「配置管理」から、日別の配置状況を一覧で確認できます。

【スクリーンショット: 配置管理ページ】

### 7.1 画面構成

#### サマリーカード
画面上部に当日の配置状況サマリーが表示されます：
- **必要人数**: 全案件の必要人数合計
- **配置済み**: 配置済みスタッフ数合計
- **不足**: 不足人数（不足時は赤、超過時は緑で表示）

#### 案件テーブル
全案件が一覧表示されます：

| 項目 | 説明 |
|------|------|
| 時間 | 上棟/終日/AM/PM/夜勤/未定（クリックでソート） |
| 顧客 | 顧客の名前 |
| 現場名 | 作業現場 |
| 住所 | 現場住所 |
| 配置スタッフ | 配置済みスタッフ名一覧 |
| 必要人数 | 案件の必要人数 |
| 配置人数 | 配置済み人数 |
| 過不足 | 不足/余裕の人数（バッジ表示） |

#### フィルタ機能
- **時間区分**: 上棟/終日/AM/PM/夜勤/未定で絞り込み
- **ステータス**: 未配置/配置済/保留/完了で絞り込み

#### ソート機能
「時間」ヘッダーをクリックすると、時間区分順（上棟→終日→AM→PM→夜勤→未定）でソートできます。再度クリックで昇順/降順を切り替えられます。

### 7.2 配置の追加・変更

1. 案件の行をクリック
2. 配置モーダルが開きます（→ 4章参照）

### 7.3 日付の切り替え

- **< 前日**: 前日の配置状況を表示
- **翌日 >**: 翌日の配置状況を表示
- **今日**: 本日に戻る

### 7.4 ダッシュボードとの違い

| 機能 | ダッシュボード | 配置管理ページ |
|------|---------------|---------------|
| 表示形式 | カード（時間帯別カラム） | テーブル（一覧） |
| 配置スタッフ表示 | なし | あり |
| 案件登録 | 可能 | 不可（閲覧のみ） |
| 用途 | 案件登録・概要確認 | 配置状況の詳細確認 |

## 8. 顧客管理

顧客情報を登録・管理します。

【スクリーンショット: 顧客一覧画面】

### 8.1 顧客の検索・絞り込み

- **検索**: 会社名・担当者名で検索
- **無効顧客表示**: チェックで無効な顧客も表示
- **単価タイプ**: 表示する単価の種類を選択

### 8.2 顧客の登録

1. 画面右下の「**+**」ボタン（丸いボタン）をクリック
2. 顧客情報を入力
3. 「**保存**」ボタンをクリック

#### 入力項目

**基本情報**

| 項目 | 必須 | 説明 |
|------|:----:|------|
| 会社名 | ○ | 会社名 |
| 支店・営業所名 | - | 支店や営業所の名称 |
| 部署名 | - | 部署名 |
| 担当者名 | - | 主な連絡先担当者 |
| 敬称 | - | 様/御中 |

**連絡先**

| 項目 | 説明 |
|------|------|
| 郵便番号 | 顧客の郵便番号 |
| 住所 | 顧客の所在地 |
| 電話番号 | 連絡先電話番号 |
| FAX番号 | FAX番号 |
| メールアドレス | 連絡先メール |

**単価設定（税抜）**

| 項目 | 説明 |
|------|------|
| 鳶単価 | 鳶作業の単価 |
| 揚げ単価 | 揚げ作業の単価 |
| 鳶揚げ単価 | 鳶揚げ作業の単価 |
| ハーフ単価 | 半日作業の単価 |
| 終日単価 | 終日作業の単価 |

**請求設定**

| 項目 | 説明 |
|------|------|
| 締め日 | 10/15/20/25/末日 |
| 支払月 | 当月/翌月/翌々月 |
| 支払日 | 10/15/20/25/末日 |
| 請求書書式 | 様式1/2/3/頭紙 |
| 消費税率 | デフォルト10% |

**その他**

| 項目 | 説明 |
|------|------|
| 状態 | 有効/無効 |
| 備考 | 特記事項 |

### 8.3 顧客の編集・削除

1. 一覧から顧客をクリック
2. 情報を修正して「**保存**」、または「**削除**」

## 9. スタッフ管理

スタッフ情報を登録・管理します。

【スクリーンショット: スタッフ一覧画面】

### 9.1 スタッフの検索・絞り込み

- **検索**: 氏名・電話番号・住所で検索
- **種別フィルター**: 自社/学生/個人事業主/外注で絞り込み
- **日給タイプ**: 表示する日給の種類を選択
- **無効スタッフ表示**: チェックで無効なスタッフも表示

### 9.2 スタッフの登録

1. 画面右下の「**+**」ボタン（丸いボタン）をクリック
2. スタッフ情報を入力
3. 「**保存**」ボタンをクリック

#### 入力項目

**基本情報**

| 項目 | 必須 | 説明 |
|------|:----:|------|
| 氏名 | ○ | スタッフの氏名 |
| 氏名カナ | - | 氏名のフリガナ |
| 種別 | ○ | 自社/学生/個人事業主/外注 |
| 外注先 | - | 外注の場合の所属会社 |
| スキル | - | 鳶/揚げ/鳶揚げ/ボード（複数選択可） |

**連絡先**

| 項目 | 必須 | 説明 |
|------|:----:|------|
| 電話番号 | - | 連絡先電話番号 |
| LINE ID | - | LINEのID |
| 緊急連絡先 | - | 緊急時の連絡先 |
| 郵便番号 | - | 住所の郵便番号 |
| 住所 | - | 現住所 |

**日当設定**

| 項目 | 説明 |
|------|------|
| 鳶 | 鳶作業の日当 |
| 揚げ | 揚げ作業の日当 |
| 鳶揚げ | 鳶揚げ作業の日当 |
| ハーフ | 半日作業の日当 |
| 終日 | 終日作業の日当 |

**個人情報**

| 項目 | 説明 |
|------|------|
| 生年月日 | スタッフの生年月日 |
| 性別 | 男性/女性/その他 |
| 血液型 | A/B/O/AB/不明 |
| バイク保有 | あり/なし |
| 雇入日 | 雇用開始日 |
| 職種 | 職種名（例: 鳶職） |

**資格・保険情報**

| 項目 | 説明 |
|------|------|
| 技能者ID (CCUS) | 建設キャリアアップシステムID |
| 雇用保険番号(下4桁) | 雇用保険の下4桁 |
| 健康保険 | 加入している健康保険の種類 |
| 年金 | 加入している年金の種類 |
| 建退共 | 建設業退職金共済の加入状況 |
| 中退共 | 中小企業退職金共済の加入状況 |

**その他**

| 項目 | 説明 |
|------|------|
| 状態 | 稼働中/停止中 |
| 外国人区分 | 外国人の場合の区分 |
| NG顧客 | 配置不可の顧客（カンマ区切り） |
| 備考 | 特記事項 |

### 9.3 スタッフの編集・削除

1. 一覧からスタッフをクリック
2. 情報を修正して「**保存**」、または「**削除**」
