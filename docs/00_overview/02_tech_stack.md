# 2. 技術スタック

## 2.1 プラットフォーム構成

| レイヤー | 技術 | 備考 |
|----------|------|------|
| プラットフォーム | Google Workspace Standard | 既存契約を利用 |
| データベース | Google スプレッドシート | シートをテーブルとして使用 |
| バックエンド | Google Apps Script (GAS) | V8ランタイム使用 |
| フロントエンド | HTML / CSS / JavaScript | GAS Webアプリとして配信 |
| ファイル管理 | Google ドライブ | PDF出力先・顧客関連ファイル格納 |

## 2.2 技術的制約

| 制約項目 | 制限値 | 対策 |
|----------|--------|------|
| GAS実行時間 | 最大6分/実行 | 処理分割・キャッシュ活用 |
| スプレッドシート | 1,000万セル/シート | 年度別シート分割 |
| 同時編集 | 競合が起きうる | `expectedUpdatedAt` による楽観ロック（競合検知＋警告表示） |
| 対応ブラウザ | Chrome最新版のみ | 他ブラウザは動作保証外 |
| 対応端末 | PCのみ | スマホ・タブレット対象外 |

## 2.3 外部ライブラリ方針

**原則として外部ライブラリは使用しない**

引き継ぎ容易性を優先し、GAS標準機能のみで実装する。

検討対象（必要に応じて）:
- Moment.js（日付処理）: GAS版を組み込み
- PDFKit相当機能: GAS標準のPDF生成機能を使用
