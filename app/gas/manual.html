1. はじめに

1.1 システム概要

配置管理システムは、日々の案件に対してスタッフを配置し、連絡用のLINEテンプレートを作成するためのWebアプリケーションです。

1.2 本マニュアルの範囲

  • ダッシュボード（案件カード表示）
  • 案件の登録・編集
  • スタッフの配置
  • LINEテンプレート
  • 設定（顧客・スタッフ・外注先・交通費・自社情報）
ℹ️ 請求・支払について
請求・支払に関する操作は「請求・支払マニュアル」（本ドキュメントのタブ2）を参照してください。

1.3 推奨ブラウザ

Google Chrome（最新版）

2. クイックスタート

5分でわかる基本操作です。

案件を登録してスタッフを配置する

  1. ダッシュボードを開く
  2. 右下の「＋」ボタンをクリック → 案件情報を入力 → 「保存」
  3. 作成されたカードをクリック → 配置モーダルが開く
  4. 左のスタッフ一覧から配置したいスタッフを選択
  5. 「変更を登録」をクリック

配置済みの案件を確認する

  1. ダッシュボードで日付を選択
  2. ステータスフィルタで「配置済」を選択
  3. カードで配置状況を確認

3. ダッシュボード

ダッシュボードは本システムのメイン画面です。選択した日付の案件を時間区分別の6列グリッドでカード表示します。

3.1 画面構成

  • 日付ナビゲーション: <（前日）/ >（翌日）/ 「今日」ボタン / 日付クリックでカレンダー選択
  • ステータスフィルタ: すべて 未配置 配置済 保留 キャンセル 問題あり
  • 統計表示（右側）: 総件数 / 配置済 / 未配置

3.2 6列グリッドレイアウト

案件は時間区分ごとに6つの列に分類されて表示されます。

上棟
件数 / 名数 / 過不足
終日
件数 / 名数 / 過不足
AM
件数 / 名数 / 過不足
PM
件数 / 名数 / 過不足
夜勤
件数 / 名数 / 過不足
未定
件数 / 名数 / 過不足

各列のヘッダーには以下が表示されます：

表示意味例
件数その時間区分の案件数3件
名数配置済/必要人数3/5名
過不足人員の過不足2名不足 / +1名余裕 / 過不足なし

3.3 案件カード（表示内容）

各カードには以下の情報が表示されます。

  • 作業カテゴリタグ（上棟 軽作業 荷揚げ）
  • ステータスバッジ（保留時のみ表示）
  • 顧客名
  • 配置人数（配置済/必要人数）— 不足時は赤色で表示
  • 鉛筆アイコン（マウスホバー時に表示）
  • 現場名・現場住所
  • 開始時間（設定されている場合）
  • 配置済みスタッフの名前チップ（最大4文字で表示）
ℹ️ 人数不足の強調
人数不足の案件カードには赤い左ボーダーが付き、一目で人員不足がわかるようになっています。

3.4 案件カード（操作）

  • カードをクリック: 配置モーダルを開く
  • 鉛筆アイコンをクリック: 案件編集モーダルを開く

3.5 その他の機能

  • フローティング「＋」ボタン: 画面右下に固定表示。クリックで新規案件の登録
  • 自動更新: 30秒ごとに最新データを自動取得。別タブでの変更もリアルタイムに反映されます
  • 印刷対応: ブラウザの印刷機能でダッシュボードの内容を印刷できます

4.7 編集・削除

  • 編集: カードの鉛筆アイコンをクリック → 編集モーダルで内容を変更
  • 削除: 編集モーダル内の「削除」ボタン
⚠️ 削除に関する注意
削除した案件は復元できません。削除前に内容を確認してください。
⚠️ アーカイブデータの編集
過去年度のデータを編集する場合、画面上部に警告バナーが表示されます。編集は可能ですが、請求書の再生成が必要になる場合があります。

5. スタッフの配置

5.1 配置モーダルを開く

ダッシュボードのカードをクリックすると、配置モーダルが開きます。

5.2 ヘッダー情報

モーダル上部に案件の概要が表示されます。

  • 顧客名・現場名・現場住所
  • 作業カテゴリタグ・作業内容
  • 作業日・時間区分・開始時間
  • 統計: 必要人数 / 配置人数 / 不足人数
⚠️ 請求集計済み
この案件の請求書が集計済みの場合、警告バナーが表示されます。配置を変更すると請求書の再集計が必要になります。

5.3 左パネル（スタッフ一覧）

  • 名前検索: テキスト入力でスタッフを絞り込み
  • スキルフィルタ: 鳶／ボード／揚げ など（スタッフマスタで設定したスキル）
  • 種別フィルタ: 全タイプ／自社／学生／一人親方／外注
  • 配置済みスタッフは上部に表示（背景色で区別）
⚠️ 重複配置の警告
同日の別案件に既に配置されているスタッフには警告マークが表示されます。配置は可能ですが、確認してください。

5.4 右パネル（一括配置設定）

5.6 配置の解除

  1. 配置済みスタッフの「×」をクリック
  2. 「変更を登録」をクリック

8. よくある質問（FAQ）

配置について

配置を間違えた場合は？

配置モーダルで「×」をクリックして解除し、正しいスタッフを選択して再配置してください。

同じスタッフを複数の案件に配置できますか？

可能ですが、同日の別案件に配置済みの場合は警告マークが表示されます。

詳細モードの「枠」とは何ですか？

1つの案件内で異なる時間区分・単価区分のスタッフを管理するための区分です。例えば、午前と午後で異なる単価を設定したい場合に使用します。

案件について

過去の案件を編集できますか？

ダッシュボードで日付を過去に移動して編集できます。ただし、請求書が生成済みの場合は請求書の再生成が必要です。

案件を誤って削除してしまった場合は？

削除した案件は復元できません。再度登録してください。

9. 運用メモ・注意点

  • 案件の詳細変更（配置内容・単価など）は、請求書側の再集計が必要になる場合があります（請求側マニュアル参照）。
  • 詳細モードで作成した案件は、配置時に「配置枠」を必ず選択してください。
  • 人数不足の案件は赤いボーダーで強調表示されます。優先的に対応してください。
  • ダッシュボードは30秒ごとに自動更新されます。手動でリロードする必要はありません。

2. クイックスタート

請求書作成から送付までの基本フローです。

配置データ → 請求書集計 → 確認・編集 → 帳票出力 → 送付 → 入金記録 ↑ 変更が必要な場合は配置データを修正して再集計

  4. 発行日・支払期限を入力 → 「PDF」で出力

3.6 配置変更の警告（⚠️）

⚠️ 配置変更マーク
請求書に紐づく配置が変更された場合、一覧に ⚠️ が表示されます。クリックすると再生成の案内が表示されます。

  • 顧客フォルダが未作成の場合: 「作成」ボタンが表示されます
  • 作成済みの場合: 「フォルダ」リンクが表示されます（クリックでGoogle Driveを開く）

3.8 請求書の集計

個別集計

  1. 「+ 請求集計」をクリック
  2. 顧客（オートコンプリート）・対象年月を選択
  3. 「集計」をクリック
💡 確認方法
一覧に新しい請求書が表示され、ステータスが「未送付」になっていれば成功です。
ℹ️ 既存の請求書がある場合
同じ顧客・年月の請求書が既にある場合は、「上書き集計」の確認が表示されます。

一括集計

  1. 「一括集計」をクリック
  2. 対象年月を選択
  3. 既存請求書の扱い（スキップ／上書き）を選択
  4. 必要に応じて「自動で続ける」にチェック（最後まで連続実行）
  5. 「開始」をクリック

プログレスバーと経過時間が表示されます。大量の顧客がある場合は「次の50社」ボタンで段階的に実行することもできます。

締め日と請求対象期間

請求書は顧客マスタの「締め日」設定に基づいて対象期間を自動計算します。

3.9 請求書の編集／帳票出力

  1. 一覧の「確認」（未送付）または「帳票出力」ボタンをクリック
  2. 請求書編集モーダルが開きます
  3. 必要に応じて以下を入力：発行日 / 支払期限 / 社内メモ
  4. 帳票を出力：「PDF」「PDF+頭紙」「Excel」「頭紙のみ」

💡 ヒント
「確認」ボタンからモーダルを開くと、PDFを生成せずに内容を事前確認できます。内容に問題がないことを確認してから出力すると、やり直しの手間を減らせます。
⚠️ 編集制限

⚠️ 送付済の編集
送付済・未回収・入金済の請求書を編集したい場合は、先にステータスを「未送付」に変更してください。

様式3の「PDF+頭紙」は横印刷専用です。縦印刷の頭紙が必要な場合は「頭紙のみ」で個別に出力してください。

3.10 入金記録

  1. 一覧の「入金」ボタンをクリック
  2. 入金日・入金額・入金方法（銀行振込/現金/その他）を入力
  3. 必要に応じて詳細オプション（銀行参照番号・備考）を入力
  4. 「入金を記録」をクリック

入金履歴はモーダル内のテーブルで確認できます。誤った入金記録は「取消」ボタンで取り消せます。

3.11 ステータス変更（一括操作）

  1. 一覧で請求書にチェックを入れます
  2. 画面下部のスティッキー一括操作バーでステータスを選択（未送付/保留/送付済/未回収）
  3. 「適用」をクリック
ℹ️ 大量操作時
100件を超える場合は自動的に分割送信されます。進捗表示（例：更新中... (1/3)）が表示されるので、完了までお待ちください。

4. 帳票一括出力とURL共有

4.1 この章でできること

  • 複数の請求書をまとめて出力する
  • URL共有の設定を理解する
  • 送信用CSVを取得する

4.2 帳票一括出力

  1. 一覧で請求書にチェックを入れます
  2. 一括操作バーの「帳票一括出力」ボタンをクリック
  3. 出力形式を選択: PDF / PDF+頭紙（送付状付き）/ Excel（編集可能）
  4. 必要に応じてURL共有のチェックを確認
  5. 「出力開始」をクリック

出力ファイルの保存先

一括生成された帳票は、Google Drive上のそれぞれの顧客専用フォルダに保存されます。既存ファイルがある場合は上書きされます。

4.3 顧客への共有方法（おすすめ）

💡 推奨
印刷郵送以外の場合は、ファイルそのものを添付するより、URL共有をおすすめします。

4.4 URL共有オプション

URL共有のメリット

  • メール添付サイズの制限を気にしなくて良い
  • 最新版を常に参照できる
  • 送信後の誤りに気付いた場合も差し替えが可能

初期状態ではURL共有がONです。リンクを知っている人は誰でもPDFを閲覧できます。

💡 使い方のヒント
  • メールでリンクを送信する際は、宛先アドレスを確認してから送信してください
  • 送信後にファイルを更新すると、同じリンクから最新版が閲覧できます
  • URL共有が不要な場合は、チェックを外してから出力してください
⚠️ 注意
ファイル添付と同様に、送信先には注意してご利用ください。

5. 支払管理（スタッフ向け）

5.1 この章でできること

  • スタッフの未払を集計する
  • 支払履歴を確認する
  • 確認済・支払済の処理を行う
  • 振込集計を出力する

5.2 画面構成

支払画面は2つのタブで構成されます。

  • スタッフ支払タブ（本章で説明）
  • 外注費管理タブ（次章で説明）

各タブの上部に「集計エクスポート」「📁 出力先」ボタンがあります。

ビュー切替

スタッフ支払タブでは2つのビューを切り替えられます：

  • 未払い集計ビュー: スタッフごとの未払い金額を集計表示
  • 支払履歴ビュー: 過去の支払記録を期間検索で表示

5.3 未払集計の手順

  1. 「スタッフ」から対象スタッフを選択（全スタッフ or 個別）
  2. 「締日」を入力
  3. 「集計」をクリック
💡 確認方法
未払サマリーカードに合計が表示されれば成功です。サマリーは「▼」で開閉できます。

5.4 ステータス変更（一括操作）

一括操作バーから以下を選択可能です。行単位の操作／一括操作の両方に対応しています。

確認済みにする 支払済にする 未払に戻す

5.5 未払詳細

  1. 一覧のスタッフ行をクリック
  2. 未払詳細モーダルが開き、出勤現場一覧が表示されます
  3. 必要なら「支払済にする」をクリック
ℹ️ 単価区分の表示
単価区分が未設定の場合は「基本」と表示されます。

詳細モーダルからは「明細出力」ボタンで個別の明細を出力することもできます。

人工割調整について

⚠️ 人工割調整
過剰配置（必要人数を超える配置）がある場合、支払額が按分調整されます。未払詳細にはオレンジ色の「人工割調整」行が表示され、調整額が確認できます。この調整額は支払エクスポートにも反映されます。

5.6 支払履歴ビュー

ビュー切替で「支払履歴」を選択し、期間（開始日〜終了日）を指定して検索します。過去の支払記録を一覧で確認できます。

5.7 振込集計エクスポート

  1. 「集計エクスポート」をクリック
  2. 期間（開始日／終了日）を指定
  3. 「エクスポート」をクリック

同名ファイルが存在する場合は「上書き」または「別名保存」を選択できます。

8. よくある質問（FAQ）

請求書について

送付済の請求書を編集したい場合は？

ステータスを「未送付」に変更してから編集してください。編集後、再度「送付済」に変更してください。

請求書の金額を変更したい場合は？

金額は配置データから計算されるため、直接編集できません。配置データを修正してから請求書を再生成してください。ただし、材料費・値引きなどの調整項目は編集モーダルから追加できます。

請求書が集計されない場合は？

対象月に配置データがない可能性があります。ダッシュボードで該当月の配置状況を確認してください。

帳票出力について

PDF+頭紙を縦印刷したいのですが？

様式3（ポラテック向け）は横印刷専用です。縦印刷の頭紙が必要な場合は「頭紙のみ」で個別に出力してください。

一括出力したファイルはどこに保存されますか？

Google Drive上の各顧客の専用フォルダ（「請求書」フォルダ）に保存されます。

URL共有について

URL共有を後から無効にできますか？

Google Driveの共有設定から無効にできます。顧客フォルダを開き、対象ファイルの共有設定を変更してください。

11. 運用メモ・注意点

請求書編集について

  • 明細の金額・数量・単価を変更したい場合は、配置データを修正して再集計してください。
  • 材料費・値引きなどの調整は、編集モーダルの「調整項目」から追加できます。
  • 送付済の請求書を編集する場合は、ステータスを「未送付」に変更してから編集してください。

URL共有について

  • URL共有は誤送信リスクがあるため、運用ルールを明確にしてください。
  • 印刷郵送以外の場合は、ファイル添付よりURL共有を推奨します。

様式3（ポラテック）について

  • 様式3の「PDF+頭紙」出力は横印刷です。縦印刷の頭紙には対応していません。
  • 縦印刷が必要な場合は「頭紙のみ」で個別出力してください。

その他

  • 出力フォルダ未設定の場合、エクスポートは実行できません。
